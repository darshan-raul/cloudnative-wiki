# Domain 4: Accelerate Workload Migration and Modernization

This domain constitutes **20%** of the exam and focuses on moving workloads to AWS and modernizing them.

## Key Areas
*   **4.1 Migration Assessment**: Strategies and tools for assessment.
*   **4.2 Migration Approach**: Data, database, and server migration.
*   **4.3 New Architecture for Migrated Workloads**: Selection of compute and storage services.
*   **4.4 Modernization and Enhancements**: Decoupling, serverless adoption, and database modernization.
