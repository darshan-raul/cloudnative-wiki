Helm Commands

Complete reference for Helm CLI commands. Run helm <command> --help for detailed help.

Installation & Upgrades

helm install

Install a chart into Kubernetes.

helm upgrade

Upgrade a release to a new chart version.

helm rollback

Roll back a release to a previous revision.

helm uninstall

Uninstall a release from Kubernetes.

Release Management

helm list

List all releases in a namespace.

helm status

Display the status of a named release.

helm history

Fetch release history.

helm get

Download extended information for a named release.

Chart Operations

helm create

Create a new chart with the given name.

helm package

Package a chart directory into a chart archive.

helm lint

Examine a chart for possible issues.

helm template

Locally render templates.

helm diff (requires helm-diff plugin)

Show differences between chart versions.

Repository Management

helm repo add

Add a chart repository.

helm repo update

Update information of available charts locally.

helm repo list

List chart repositories.

helm repo remove

Remove one or more chart repositories.

helm repo index

Generate an index file from a directory of charts.

Search

helm search

Search for charts.

Dependency Management

helm dependency build

Rebuild the charts/ directory based on Chart.lock.

helm dependency update

Update charts/ based on Chart.yaml.

helm dependency list

List dependencies for a chart.

Testing

helm test

Run tests for a release.

Registry Operations

helm registry login

Login to a registry.

helm registry logout

Logout from a registry.

helm push

Push a chart to OCI registry.

Plugin Management

helm plugin install

Install Helm plugins.

helm plugin list

List installed plugins.

helm plugin update

Update plugins.

helm plugin uninstall

Uninstall plugins.

Environment & Configuration

helm env

Print Helm client environment information.

helm version

Print version information.

helm completion

Generate autocompletion scripts.

Verification

helm verify

Verify that a chart at the given path has been signed and is valid.

helm plugin verify

Verify that a plugin is signed and valid.

Global Flags

FlagDescription
--debugEnable verbose output
--kube-contextSet the kube context
--namespace / -nSet the namespace
--kubeconfigPath to kubeconfig file
--dry-runSimulate operations
--timeoutSet timeout (Go duration format)
--waitWait for resources to be ready
--no-hooksSkip running hooks
--skip-schema-validationSkip schema validation
--setSet values
--set-fileSet values from file
--set-stringSet string values
--values / -fSet values from file
